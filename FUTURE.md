# Nexus - Future Enhancements

## Potential Features for Future Versions

### 1. Task Tags/Labels
- Add custom tags to tasks (e.g., #work, #personal, #urgent)
- Filter tasks by tags
- Color-coded tag display

### 2. Recurring Tasks
- Daily, weekly, monthly task templates
- Auto-create tasks on schedule
- Carry over incomplete recurring tasks

### 3. Pomodoro Timer
- Built-in 25-minute focus timer
- Break reminders
- Session tracking per task

### 4. Task Duration Tracking
- Track total time spent on each task
- Display cumulative time in timeline
- Daily/weekly time reports

### 5. Cloud Sync (Optional)
- Sync data to a backend service
- Multi-device support
- Conflict resolution

### 6. Polish & UX
- Smooth animations using harmonica library
- Sound notifications for timer completion
- Custom keybinding configuration file
- Task notes/descriptions with expand view

### 7. Testing
- Unit tests for domain logic (task, timeline, calendar)
- Integration tests for storage layer
- End-to-end TUI tests

### 8. Distribution
- Create releases via goreleaser (v0.1.0)
- Pre-built binaries for:
  - Linux (amd64, arm64)
  - macOS (amd64, arm64)
  - Windows (amd64)
- Homebrew formula
- AUR package

### 9. Additional Themes (Optional)
- Terminal Classic (green/amber on black)
- Minimal (grayscale)
- High Contrast (accessibility)

### 10. Advanced Features
- Task dependencies (blocked by)
- Task templates
- Bulk operations (multi-select)
- Archive completed tasks
- Statistics dashboard
